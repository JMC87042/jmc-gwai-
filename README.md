# 🐋 JMC Global Whale Intelligence

TikTok Live Whale Analytics Platform

## Features
- Real-time whale tracking
- REST API
- SQLite database
- Responsive UI

## Endpoints
- GET /api/whales/ranking
- GET /api/hosts/ranking
- GET /api/analysis/timeline

- API: PythonAnywhere
